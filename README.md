# Airbnb Room Availability Analysis & Prediction

## Project Overview
This project analyzes Airbnb listing data and builds a machine learning model to predict room availability over a year.

## Objectives
- Understand factors affecting Airbnb availability
- Perform exploratory data analysis (EDA)
- Build a predictive model using machine learning

## Dataset
The dataset includes:
- Listings data (property details)
- Calendar data (availability)
- Reviews data (customer feedback)

## Key Steps
1. Data Cleaning and Preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature Engineering  
4. Machine Learning Model (Random Forest)  
5. Model Evaluation  

## Results
- Built a Random Forest model to predict availability
- Evaluated performance using MAE and R²
- Visualized predictions and feature importance

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Conclusion
The project demonstrates how listing characteristics influence Airbnb availability and shows how machine learning can be used to predict it.
